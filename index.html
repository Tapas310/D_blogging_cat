Struggles

7/22/23
Ok, ok I dont always drop rhymes and poems but.. I would like to talk about some struggles with learning to program. Its tough! and I thought digging through my neighbor's garbage was hard. Nah fam.

Real talk doe
I am currently following App Academy's Online Software Engineering Program curriculum. Honestly, I love it. It's very clear, It has reading material, step-by-step instructions and even videos. That being said, my personal experience learning HTML and CSS has been good! However, my main challenge has been JavaScript.

JS is so fun and interesting, therefore It's a shame that its been a bit tough for me to puzzle at times. For example, loops within functions featuring Arrays. It's slowly registering though, which is normal for some of us, so I hear. Problem solving is exactly that, it's not easy otherwise would it truly be a problem?

When the dust settles, I still absolutely love programming and It's only the beginning.
